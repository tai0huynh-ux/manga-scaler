# Roadmap

## Trace roadmap

- Trace Core MVP -- Done.
- Trace Dashboard -- Planned.
- Trace Artifact Capture -- Planned.
- Reproduction Package -- Planned.

## Next validation

- Run the extension on long manga chapters from multiple hosts and record discovered images, raw slices, enqueued segments, and completed segments.
- Add site-specific fixtures only when runtime evidence demonstrates a repeatable incompatibility.
- Consider a local original-preview cache only if placeholder-only previews are insufficient; avoid introducing IndexedDB preview storage without a measured need.

## Out of current scope

- Canvas/WebGL image discovery.
- Streaming segment crop/enqueue.
- Site-specific authentication integrations.
